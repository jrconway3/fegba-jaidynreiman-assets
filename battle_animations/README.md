## Mounted

### Wolf Knight

Wolf Knight was made by VelvetKitsune using the Mouth Douge as a base, which VelvetKitsune created a standard wolf out of, then the Wolf Knight.

I made a few fixes to some of the original Wolf Knight anims as well as adding a brand new Bow animation. Technically speaking both versions are considered Female, but I opted to call the short haired version as Male.

I wanted a long hair version as I tend to really like long hair. Because the character is leaning forward a bit in many frames I have the long hair in both the front and back a bit, but I adjusted it in some places to make it cleaner. In the end I opted to label this one specifically as Female and the first as Male, but it doesn't matter much because I'll fully create new ones.
