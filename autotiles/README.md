# Autotiles

Autotiles are basically just map animations in FE Builder. These are things like animated water. The way autotiles work is that the first frame of the tile must exactly match a 16x16 square in an LT tilemap or tileset.

These autotiles were created by me. I created them from the original tilesets in the FE-Repo using the Map Animations that exist there, as such, I will be crediting the original artists for these map animations as well as the base tilesets.

## Cynon's Metropolis

I created autotiles for 0.8, 0.83, and 0.94 of Cynon's Metropolis.

The autotiles for the darker water do not work correctly, they are recolored blue instead. Perhaps in a future version these can be fixed. If you only use one color water (light or dark) in a single tilemap these should work.

### Streetlamp Palette Animations

I manually created every streetlamp palette animation as autotiles.

Unforunately, the autotiles for the lanterns do not work correctly. Autotiles do not seem to work well with palette shift animations, as instead of working they simply fill out all animations with the same color. You can make them work with some manual tweaking, but its not easy.
